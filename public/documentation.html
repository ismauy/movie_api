myFlix Project

The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Request Url Request body format Response format
GET /movies Return list of JSON movie objects.
GET /movies/:title Return JSON movie object.
GET /genres/:name Return JSON genre object.
GET /directors/:name Return JSON director object.
POST /users {name: '', email: ''} Return JSON user object with ID.
PATCH /users/:id/favorites/:title Return TEXT "movie added".
PUT /users/:id {id: 0, name: '', email: ''} Return JSON user object.
DELETE /users/:id/favorites/:title Return TEXT "movie removed".
DELETE /users/:id Return TEXT "user 'email' has been removed".
